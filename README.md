# SPM_Lab

A software project management laboratory repository.

## Project Structure

```
SPM_Lab/
├── README.md          # Project documentation
├── file.txt           # Feature 1 content
├── greeting.txt       # Greeting messages
├── hello.txt          # Hello messages
└── hi.txt             # Hi messages
```

## Branches

- `feat1` - Feature development branch
- `qwen-code-*` - Current working branch

## Version

Current version: v1.0

## Files Description

| File | Description |
|------|-------------|
| `file.txt` | Contains feature-specific content |
| `greeting.txt` | Contains greeting messages from different branches |
| `hello.txt` | Contains hello messages from main and feature branches |
| `hi.txt` | Contains changes from feature branch |

## Usage

This repository appears to be used for software project management exercises, demonstrating:
- Git branching strategies
- Feature development workflows
- Merge operations between branches

## Getting Started

1. Clone the repository
2. Check available branches: `git branch -a`
3. Switch to a specific branch: `git checkout <branch-name>`
